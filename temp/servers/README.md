## servers/

Один файл = один сервер. Имя файла совпадает с именем хоста в inventory.

Каждый файл описывает только то, что делает этот сервер уникальным:
- какие роли на нём запускаются
- переменные для этих ролей

Общее для всех серверов — в `servers_main.yml` в корне, не здесь.
