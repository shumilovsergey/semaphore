## Структура Ansible

```
temp/
├── servers_main.yml        — роли и переменные для всех серверов
├── servers/
│   └── <hostname>.yml      — роли и переменные конкретного сервера
├── roles/
│   ├── role_template/      — шаблон для создания новой роли
│   └── <role_name>/        — одна роль = одна задача

```

## Где что живёт

| Вопрос | Ответ |
|---|---|
| Это нужно всем серверам? | `servers_main.yml` |
| Это нужно одному серверу? | `servers/<hostname>.yml` |
| Как это делается? | `roles/<name>/` |
